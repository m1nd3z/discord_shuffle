# Discord Team Shuffle Bot

Discord botas, kuris suskirsto balso kanalo žaidėjus į komandas.

## Komandos

- `/teamshuffle` - Suskirsto žaidėjus į komandas pagal pasirinktą dydį

## Reikalavimai

- Python 3.11+
- Discord bot token

## Aplinkos kintamieji

- `DISCORD_TOKEN` - Discord bot token

## Paleidimas

```bash
python bot.py
```
